About Check

LoginCheck is an advanced browser extension that provides real-time protection against Microsoft 365 phishing attacks.


LoginCheck is based on the incredible work of the CyberDrain Team and is redistributed under the AGPL Licence. Source Code available on GitHub

Rejuvenate IT have deployed this Extension to your browser in order to safeguard your Work Login Credentials..

You may remove this Extension if the device is no longer used for Work Purposes.
